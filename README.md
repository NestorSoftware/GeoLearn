# GeoLearn 🌍

Plataforma educativa web de Historia y Geografía desarrollada como proyecto final del Grado Superior en Desarrollo de Aplicaciones Web (DAW).

## 🚀 Tecnologías

- **PHP 8** — Arquitectura MVC personalizada (sin framework)
- **PostgreSQL** — Base de datos relacional
- **Docker / Docker Compose** — Contenedores para desarrollo y despliegue
- **CSS** — Estilos propios
- **Git** — Control de versiones con flujo main/develop/feature
## 📁 Estructura
GeoLearn/
├── src/          # Código fuente (MVC)
├── database/     # Scripts SQL
├── docs/         # Documentación del proyecto
├── Dockerfile
└── docker-compose.yml
## ⚙️ Instalación

1. Clona el repositorio
```bash
   git clone https://github.com/NestorSoftware/GeoLearn.git
```
2. Levanta los contenedores
```bash
   docker-compose up -d
```
3. Accede en el navegador a `http://localhost`
